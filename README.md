# 批量创建简易CAD（dwg）格式图纸

#### 介绍
根据Excel表格中的数据，批量生成矩形、圆形和环形的简单CAD图形（dwg格式）。仅仅适用于板材切割领域（火焰切割、等离子切割、激光切割）的简易图形的批量出图，应用范围极窄。
